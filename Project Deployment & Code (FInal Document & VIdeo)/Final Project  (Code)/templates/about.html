Exploratory Data Analysis on Indian Food

This project involves conducting an Exploratory Data Analysis (EDA) on a comprehensive dataset related to Indian food. The primary focus is to gain insights into various aspects of Indian cuisine, including ingredients, recipes, regional variations, and popularity. The project encompasses the following key components:

Data Collection: Gathering a comprehensive dataset on Indian food from sources such as recipe websites, food blogs, culinary books, and public repositories. The dataset includes attributes like dish names, ingredients, preparation time, cooking time, dietary information, flavor profiles, courses, states, and regions.

Exploratory Data Analysis (EDA): Performing an in-depth analysis of the dataset, which includes: Descriptive statistics to understand the basic characteristics of the dataset. Ingredient analysis to identify commonly used ingredients and their regional variations. Data visualization to create visual representations of the data, revealing patterns and relationships. Custom analysis and insights derived from the data.

Visualization: Creating various types of visualizations, such as bar charts, line charts, pie charts, heatmaps, scatter plots, and maps to represent the findings from the EDA.

Report Generation: Compiling the results and insights from the EDA into a multi-scene report, presenting key findings, analyses, and visualizations. The report provides a holistic understanding of Indian food, including its cultural significance, culinary traditions, and regional diversity.
